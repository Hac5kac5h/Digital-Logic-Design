Decimal Conversions









  • Write the given decimal number
  • If the given decimal number is less than 2 the binary number is the same.
  • If the decimal number is greater than 2 then divide the number by 2.
  • Repeat step 3 and 4 with the quotient till it is less than 2
  • Write the remainders in reverse order (bottom to top)

  • Write the given decimal number
  • If the given decimal number is less than 8 the octal number is the same.
  • If the decimal number is greater than 7 then divide the number by 8.
  • Repeat step 3 and 4 with the quotient till it is less than 8
  • Write the remainders in reverse order (bottom to top)

  • First, divide the decimal number by 16, considering the number as an integer
  • write the remainders in reverse order (bottom to top)
  • Again divide the quotient by 16 and repeat till you get the quotient value equal to zero.
  • Remember, from 0 to 9, the numbers will be counted as the same in the decimal system
  • But from 10 to 15, they are expressed in alphabetical order such as A, B, C, D, E, F and so on.
