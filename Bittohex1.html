Binary to Hexadecimal Conversions!



  • To convert binary numbers to Hexadecimal numbers, follow the below steps:

  • Take the given binary number

  • Multiply each digit by 2^(n-1) where n is the position of the digit from the decimal

  • The resultant is the equivalent decimal number for the given binary number

  • Divide the decimal number by 16

  • Note the remainder

  • Continue the above two steps with the quotient till the quotient is zero

  • Write the remainder in the reverse order

  • The resultant is the required Hexadecimal number for the given binary number
