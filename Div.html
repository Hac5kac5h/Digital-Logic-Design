Binary Division






The binary Division operation works similarly to the base 10 decimal system, except that it is a base 2 system.

The binary system consists of only two digits, 1 and 0

  • 0 / 0 = NaN

  • 0 / 1 = 0

  • 1 / 0 = NaN

  • 1 / 1 = 1

Binary numbers are used in computers to store and represent data using digits 0 and 1. There are two cases that come up while learning about binary addition, and those are given below:

  • Binary division like decimal division



Binary division like decimal division


The division of two digits is similar to decimal division where given input is divided by dividend.
